# 🧶 Loop Log
![Crochet Logo](https://plus.unsplash.com/premium_photo-1675799559554-f2395a6afa45?q=80&w=1172&auto=format&fit=crop&ixlib=rb-4.1.0&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D)
Loop Log is a full-stack MEN (MongoDB, Express, Node.js) application that allows users to track and manage their crochet projects. Users can create, view, update, and delete projects while organizing important details like yarn, hook size, difficulty, and project status.

I built this app to create a simple and practical way to keep track of crochet projects in one place.
